# NodeMysqlApp
1- Setup Database--  Import 'test2' db in mysql .
2- Setup Project--
Install Express Generator  using Command-line - "npm install -g express-generator" . 
Create project using Command-line  -  express NodeMysqlApp  . 
Following steps- 
                 (1) cd NodeMysqlApp . 
                 (2) npm install . 
                 (3) npm start .
